# Gladiator Arena

Student libGDX project (Desktop, Java, Gradle) based on `GladiatorArena_GDD_v2.md`.

## Current Status

- Phase 1 scaffold is in place:
  - desktop launcher configured to `800x480`
  - `GladiatorGame` entry point created
  - package/class stubs added from GDD Section 11
  - `managers.AssetManager` singleton added

## Run

```bash
./gradlew lwjgl3:run
```

On Windows:

```bash
gradlew.bat lwjgl3:run
```
